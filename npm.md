# NPM Commands

### 1. To list the packages installed:

- Global packages

  ```bash
  npm ls -g --depth 0
  ```

- Local packages

  ```bash
  npm ls
  ```

- Specific local package

  ```bash
  npm ls <package_name>
  ```

### 2. To visit the link in the 'repository' field of the package.json:

```bash
npm repo
```

### 3. To initialize a repo:

- Basic use:

  ```bash
  npm init
  ```

- With all defaults:

  ```bash
  npm init --y
  ```
